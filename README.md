# 基于Java的医护人员排班系统

## 项目简介
医院医护人员排班系统是基于Java语言，Springboot框架，Mysql数据库的医护人员排班系统。角色上：1.管理员角色 2.普通医护人员角色。功能上：前台负责展示医院新、科室信息、医生信息等;后台负责管理医护人员、科室、排班等。适合作为毕业设计、大作业、课程设计等
